# Adiyogi Manish Bhagat - Premium Naturopathy Website

A world-class premium SPA designed and developed with semantic HTML, modern CSS Grid/Flexbox styling, custom animations, clean JavaScript, and Progressive Web App (PWA) support.

## Key Features
- **PWA Ready**: Features a manifest and background service-worker caching for instant offline support.
- **Glassmorphic UI**: Beautifully designed utilizing the exact requested luxury brand palette: Forest Green, Gold, White, and Bronze.
- **Dynamic Product Engine**: Features micro-instant filtering and fully-integrated automated WhatsApp booking pipelines.
- **Light/Dark Theme Support**: Optimized with native CSS properties changing color schemes globally based on system preferences or toggle actions.

## Deployment on GitHub Pages
1. Push all these files straight to a public/private GitHub Repository.
2. Navigate to **Settings** > **Pages**.
3. Under the *Build and deployment* section, select **Deploy from a branch** (deploying `main` or `master` folder `/root`).
4. Click Save. Your website will be active live within 1–2 minutes!
